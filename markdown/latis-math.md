- Basic
  - $+ + \to +$
  - $+ - \to -$
  - $- + \to -$
  - $- - \to +$
- $a(b+c) = ab + ac$
  - a 乘以括號b加c等於ab加ac
- $a(b-c) = ab - ac$
  - a 乘以括號b減c等於ab減ac
- 其他變形
  - $(b+c)=b+c$
  - $-(b+c)=-b-c$
  - $-a(b+c) = -ab-ac$
  - $-a(b-c) =-ab+ac$
- $(b+c)a = ba + ca$
- $(b+c)/a=b/a+c/a$
  - $(b-c)/a=b/a-c/a$
  - $(b-c)/(-a)=-b/a+c/a$
